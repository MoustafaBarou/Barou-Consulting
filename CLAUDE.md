# Barou Consulting

Dit project betreft de officiële website van Barou Consulting.

Domein:
https://barouconsulting.nl

Status:
Production project.

Werk alsof deze website binnen enkele weken live gaat.

Er worden geen prototypes of demo's meer gebouwd.

Alle wijzigingen worden direct in de productiecode uitgevoerd.

---

# Bedrijf

Barou Consulting is een gespecialiseerde Microsoft 365 Consultancy.

Ik ben zelfstandig Microsoft 365 Consultant.

De website mag nooit uitstralen alsof Barou Consulting een groot consultancybedrijf is.

Maar ook nooit alsof het een freelancer is met een online CV.

De juiste positionering is:

Een gespecialiseerde Microsoft 365 Consultant met een professionele consultancypraktijk.

Voordelen die centraal staan:

- Direct contact
- Korte lijnen
- Persoonlijke betrokkenheid
- Specialistische kennis
- Eigenaarschap
- Kwaliteit
- Betrouwbaarheid

---

# Doelgroep

Primair:

- ICT Beheerders
- Directeuren

- Enterprise

# Design Philosophy

De website moet voelen zoals een goed ingerichte Microsoft 365 omgeving.

Rustig.

Veilig.

Overzichtelijk.

Modern.

Professioneel.

Tijdloos.

Vorm volgt altijd functie.

Geen enkel element wordt toegevoegd puur voor decoratie.

# Motion

Gebruik Framer Motion.

Alleen subtiele animaties.

Toegestaan:

- Fade
- Reveal
- Hover
- Stagger

Niet gebruiken:

- Bounce
- Parallax
- Scroll-jacking
- Grote animaties

Respecteer altijd prefers-reduced-motion.

# Schrijfstijl

De website moet voelen alsof hij geschreven is door een ervaren Nederlandse copywriter.

Niet door AI.

## Verboden

Gebruik NOOIT de em dash.

Gebruik ook geen en dash als stijlmiddel.

Schrijf natuurlijke Nederlandse zinnen.

Gebruik:

- punten
- komma's
- dubbele punten

of begin een nieuwe zin.

## Vermijd AI-patronen

Niet gebruiken:

- Niet alleen..., maar ook...
- Het gaat niet om..., het gaat om...
- Of je nu...
- Denk aan...
- Stel je voor...
- Kortom...
- Bovendien...
- Daarnaast...
- Hier komt...

Gebruik alleen wanneer het echt natuurlijk is.

## Zinsbouw

Gebruik:

Korte zinnen.

Actieve zinnen.

Rustige toon.

Geen marketingtaal.

Geen buzzwords.

Geen holle claims.

Geen overdreven bijvoeglijke naamwoorden.

Niet schrijven:

- hoogwaardig
- innovatief
- toonaangevend
- state-of-the-art
- premium
- wereldklasse

Laat kwaliteit blijken uit de inhoud.

Niet uit de woorden.

## Correct Nederlands

Koppeltekens mogen alleen gebruikt worden wanneer de Nederlandse spelling dit vereist.

Bijvoorbeeld:

- Microsoft 365-omgeving
- Zero Trust-principes
- e-mailbeveiliging

Verder geen onnodige streepjes.
